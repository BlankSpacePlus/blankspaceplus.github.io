# 操作指南

提交的部分是项目的`public`文件夹，完成项目基于`Node.js`构建，博客框架为`Hexo`，最终定下的模板为`Matery`：

构建项目：
```shell
hexo init blankspaceplus.github.io
```

更新后运行项目：
```shell
hexo clean
hexo g -d
hexo s
```
