# Hexo搭建

- [CSDN: GitHub Pages + Hexo搭建个人博客网站](https://blog.csdn.net/yaorongke/article/details/119089190)
- [知乎: 超详细Hexo+Github博客搭建小白教程](https://zhuanlan.zhihu.com/p/35668237)

项目基于`Node.js`构建，博客框架为`Hexo`，最终定下的模板为`Matery`。

构建项目：
```shell
hexo init hexo-blog
```

更新后运行项目：
```shell
hexo clean
hexo g -d
hexo s
```
